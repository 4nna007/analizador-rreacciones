Analizador de Sentimientos con Machine Learning

Este mini proyecto implementa un sistema de análisis de reacciones utilizando técnicas de procesamiento de lenguaje natural (NLP) 
y modelos de Machine Learning.

Características

- Preprocesamiento de texto (limpieza, emoticonos, negaciones)
- Extracción de características con TF-IDF
- Entrenamiento de múltiples modelos:
  - SVM
  - Random Forest
- Evaluación con métricas:
  - Accuracy
  - Precision
  - Recall
  - F1-score
