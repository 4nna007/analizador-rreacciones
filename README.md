# 🧠 Analizador de Sentimientos con Machine Learning

Un mini proyecto robusto que implementa un sistema de análisis de sentimientos utilizando técnicas avanzadas de procesamiento de lenguaje natural (NLP) y modelos de Machine Learning. Ideal para clasificar y entender el tono emocional en textos.

---

## 🚀 Características principales

✨ **Preprocesamiento inteligente de texto**
- Limpieza y normalización de datos
- Manejo automático de emoticonos y caracteres especiales
- Procesamiento de negaciones para contexto más preciso

🤖 **Modelos de Machine Learning**
- Support Vector Machine (SVM)
- Random Forest con optimización
- Validación cruzada para mayor confiabilidad

📊 **Evaluación exhaustiva**
- Accuracy, Precision, Recall y F1-score
- Matriz de confusión y ROC-AUC
- Reportes detallados por clase

---

## ⚙️ Tecnologías utilizadas

| Tecnología | Propósito |
|-----------|----------|
| **Python** | Lenguaje principal |
